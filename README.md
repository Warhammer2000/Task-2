## Brief

Lightweight event hosting and attendance platform — organizers publish, attendees RSVP and receive digital tickets with QR codes, Checkers verify entry at the door, and the community provides post-event feedback and photos.

Built using **Lovable Pro** with Claude Code (Opus 4.7) for pre-flight planning, R-coverage mapping, and audit-driven phase gates. See [`task-2/report.md`](./task-2/report.md) for the full process write-up.
